Hallo Leo Opdenplatz,

heute beleuchten wir erweiterte Funktionen Ihres V-Servers und zeigen Ihnen, wie Sie Server und Anwendungen jederzeit im Auge behalten können.
 
 
Vertrauen ist gut, Kontrolle ist besser
 
 
Eine durchgängige Funktion Ihrer Anwendungen hat höchste Priorität? So stellen Sie sicher, dass alles reibungslos funktioniert.
Unsere Monitoring Services informieren Sie komfortabel via SMS oder E-Mail, sollte Ihr Server oder einzelne Dienste nicht mehr zu 100% zur Verfügung stehen. Mit Monitoring Basic können Sie bereits einen Dienst kostenfrei überwachen lassen.
Schauen Sie auch regelmäßig in die Logfiles Ihres Servers. Dort werden Warnungen und Fehler Ihrer Dienste und Anwendungen in Echtzeit dokumentiert. Wo Sie die Logfiles finden, wenn Sie Plesk nutzen erfahren Sie in dieser Übersicht. 

Ist Ihr Server einmal nicht mehr erreichbar, können Sie ihn über den RecoveryManager neustarten oder das Rettungssystem aktivieren. Über das Rettungssystem können Sie Ihren Server wieder erreichen und können die Logfiles überprüfen. 
 
 
Viel Erfolg mit Ihrem V-Server Linux!

Mit freundlichen Grüßen
Ihr STRATO Server Team
