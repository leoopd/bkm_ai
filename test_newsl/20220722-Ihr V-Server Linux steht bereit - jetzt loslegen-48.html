Hallo Leo Opdenplatz,

schön, dass Sie sich für einen V-Server Linux von STRATO entschieden haben. Diese E-Mail ist die erste von vier Nachrichten, die Ihnen helfen sollen Ihren V-Server und dessen Funktionen Schritt für Schritt kennenzulernen.

Gleich zu Beginn ein Tipp: Den Server-Login zur Verwaltung Ihres neuen V-Server Linux erreichen Sie ganz einfach über den STRATO Kunden-Login.
 
 
Domains & Betriebssysteme
 
 
Im Kunden-Login können Sie Domains bestellen und verwalten. Sobald eine Domain auf Ihrem V-Server eingerichtet ist, zeigt sie automatisch auf dessen IP-Adresse. Möchten Sie eine Website auf Ihrem Server betreiben, richten Sie die Domain zusätzlich in Ihrem Webserver ein.

Je nach Betriebssystem variiert die Konfiguration eines Web- oder Mailservers. Unter Mein Server erhalten Sie im Bereich Neuinstallation einen Überblick über die angebotenen Betriebssysteme. Möchten Sie nicht auf eine grafische Oberfläche verzichten, empfehlen wir eine Kombination mit Plesk. Diese Anleitung erleichtert Ihnen den Einstieg.
 
 
Unterstützung rund um die Uhr
 
 
Im STRATO Hilfe- & Kontaktbereich finden Sie Antworten auf vertragliche und produktbezogene Fragen. Hier erfahren Sie unter anderem, wie Sie sich per SSH an Ihrem Server anmelden oder Ihren Server neu starten können.
 
 
Viel Spaß beim Einrichten Ihres V-Servers!

Mit freundlichen Grüßen
Ihr STRATO Server Team
